5. What is the bug in this program?
    valueKeeper--;
    This only works when the initial value is greater than the count value. The bug is that valueKeeper is always
    decreased using valueKeeper--. If myCounter is greater than valueKeeper, the loop will never reach the target value, so the
       loop runs forever.

6. Write C# code which continuously prompts a user to input a number, terminating if the number
   is divisible by 11. Recall that the remainder operator in C# is %, so an integer stored in num is
   divisible by 11 if (num % 11) == 0.

       using static SplashKitSDK.SplashKit;
       using static System.Convert;
       int num;
       do
       {

          Write("Enter a number: ");
          num = ToInt32(ReadLine());
       }
       while ((num % 11) != 0);
       WriteLine("The number is divisible by 11. Program ended.");
